🎓 LEARNINGS & NOTES
═══════════════════════════════════════════════════════════════════════════════

✨ TEKNİK BİLGİ:

• FileSystemWatcher event'leri throttle edilmeli (çok hızlı events)
• Large file hash'lerken UI freeze yok diye async görmek lazım
• SQLite bağlantısı thread-safe değil, Dispatcher gerekli
• Theme değişimi hızlı olacak diye ResourceDictionary use ediyoruz
• Turkish characters filename'lerde sorun yaratabilir (System.IO)

🔍 DEBUGGING İPUÇLARı:

• Output window'u açık tut (Debug output)
• Performance profiler'ı kullan (large files)
• Breakpoint'ler istediğin yerlere koy
• Database browser tools (SQLite Studio)
• WPF debugger ötüz (XAML inspector)

⚡ PERFORMANCE CONSIDERATIONS:

• Async operations kaldırmak + UI thread
• ObservableCollection çok item'da yavaş → Virtual scroll?
• Large file hash'i progress göster
• Database query optimize et (index'ler)
