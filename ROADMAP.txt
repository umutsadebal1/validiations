[✓] PHASE 4: UI Layout (XAML)

  ├─ MonitorTab.xaml - Klasör seç, ListView, durdur butonu

  └─ ThemeResources.xaml - Dark/Light tema tanımları

  ├─ MainTab.xaml.cs - UI event-leri bağla (şimdilik debug'a set et)

  └─ SettingsTab.xaml.cs - Tema toggle'ı çalışır duruma getir

[✓] PHASE 6: Database Kurulumu
  ├─ SQLite bağlantı string'i App.config'e ekle
  ├─ DatabaseService'i başlat (Connection Pool)

  ├─ Her hash sonucu otomatik olarak veritabanına kaydet

  ├─ Geçmiş ListView'ını gerçek verilerle doldur

[ ] PHASE 10: File Monitor Implementation

[ ] PHASE 11: Settings & Persistence
  ├─ Tema ayarı (Local AppData'da sakla: theme.json veya Windows Registry)

  └─ Uygulama kapatıldığında ayarları kaydet

[ ] PHASE 12: Export Functionality
  ├─ Export Format ComboBox: CSV / JSON seçimi

  └─ Desktop'e kaydet ve başarılı mesajı göster

[ ] PHASE 13: Styling & Theme System
  ├─ ThemeResources.xaml - Dark/Light ResourceDictionary'ler
